# Beitragen zur VpMobile24 Integration 🚀

Vielen Dank, dass du dir die Zeit nimmst, dieses Projekt zu unterstützen! Beiträge von der Community helfen dabei, die VpMobile24-Integration für alle Nutzer von stundenplan24.de stabil und funktionsreich zu halten.

---

## 🛠 Wie du helfen kannst

### 1. Fehler melden (Bug Reports)
Wenn du einen Fehler findest, erstelle bitte ein [Issue](https://github.com/Maximilian-Andrew-Kluge/VpMobile24/issues). Nutze dabei bitte das bereitgestellte Template und achte auf folgende Punkte:
* **Logs:** Füge relevante Fehlermeldungen aus Home Assistant hinzu.
* **Kontext:** Welche Schule/Klasse ist betroffen? Welche HA-Version nutzt du?
* **Reproduzierbarkeit:** Beschreibe kurz, wie man den Fehler auslösen kann.

### 2. Feature-Anfragen
Du hast eine Idee für einen neuen Sensor oder eine Kalender-Funktion?
* Erstelle ein Issue oder diskutiere es zuerst in den [Discussions](https://github.com/Maximilian-Andrew-Kluge/VpMobile24/discussions).
* Beschreibe genau, welchen Mehrwert das Feature bietet.
