## 📌 Beschreibung

VpMobile24 ist eine Home Assistant Integration zur Verarbeitung von Stundenplan-Daten.

Sie bietet:
- automatischen Abruf von Vertretungsplänen
- strukturierte Wochen- und Tagesdaten
- Integration in Lovelace Dashboards
- optionale visuelle Card

---

## ✨ Funktionen

### Integration
- 📅 Wochenübersicht (Mo–Fr)
- 🔄 automatischer Datenabruf (15 Min Intervall)
- 👥 mehrere Klassen gleichzeitig möglich
- 📊 strukturierte Sensoren pro Tag
- ⚡ automatische Aktualisierung

### Lovelace Card
- 🎨 visuelle Konfiguration (kein YAML erforderlich)
- 📱 responsive Design (Mobile & Desktop)
- 🔥 heutiger Tag hervorgehoben
- ⛔ Vertretungen farblich markiert
- ⚙️ vollständig anpassbare Darstellung
- 🌗 Theme-Unterstützung
